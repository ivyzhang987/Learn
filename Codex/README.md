# Playground

A small static website for mini browser projects.

## Run locally

Start a static file server from the project root, for example:

```bash
python3 -m http.server 4173
```

Then open:

- `http://localhost:4173/` for the homepage
- `http://localhost:4173/snake/` for the Snake game

## Structure

- `/` is the homepage
- `/snake/` contains the Snake game

## Publish

This project can be deployed to any static host, including GitHub Pages, Netlify, or Vercel.
